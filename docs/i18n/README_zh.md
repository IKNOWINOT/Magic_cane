## 概述

Magic Cane 是一套为盲人和低视力用户设计的辅助导航系统。系统包含三个设备：

1. **智能手杖** — 主安全计算机（ESP32-S3）
2. **360°触觉腰带** — 方向性振动导航
3. **手机伴侣** — RynnBrain辅助感知（可选）

## 设计原则

- 🛡️ **手杖是安全权威。** 所有停止/前进决策都在本地使用确定性算法做出。
- 🔔 **腰带是触觉输出设备。** 接收手杖的命令，在危险方向振动。
- 🧠 **RynnBrain仅为建议性质。** 提供场景标签、OCR和路线建议，但绝不覆盖安全决策。
- 🌐 **安全路径无云依赖。** 如果手机断开连接，手杖和腰带继续正常工作。

## 手杖传感器

| 传感器 | 功能 |
|--------|------|
| 4× VL53L5CX ToF | 4个象限障碍物检测（8×8区域） |
| 1× TFmini-S 激光雷达 | 前方远距离测距 |
| 1× VL53L1X ToF（向下） | 台阶和高差检测 |
| BNO086 IMU | 方向、步态检测、倾斜 |

## 危险等级

| 等级 | 距离 | 响应 |
|------|------|------|
| 无 | ≥ 2.5 米 | 无振动 |
| 警告 | 1.2 – 2.5 米 | 轻微振动 |
| 近距离 | 0.4 – 1.2 米 | 强烈振动 |
| 停止 | < 0.4 米 | 紧急停止 |
| 跌落 | 检测到高差 | 所有振动最大 |
